# Conecta
## Descripción

Aplicación desarrollada en lenguaje C para simular el funcionamiento de una red social al estilo Twitter/X
## Requisitos

* Compilador C (gcc recomendado)
* Sistema operativo Linux, Mac o Windows (con MinGW)

## Compilación

### Windows (MinGW)
```
gcc Conecta_main.c Conecta/*.c TDAs/*.c -I Conecta -I TDAs -Wno-unused-result -o Conecta
```

## Ejecución

```
./Conecta
```
